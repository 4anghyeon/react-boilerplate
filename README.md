# 개요

React + TypeScript + TailwindCSS + Vite를 이용한 React 프로젝트 Boilerplate

```bash
pnpm create vite react-boilerplate --template react-ts
```

## 기본 설정

- eslint 적용 (.eslintrc.cjs)
- prettier 적용 (.prettierrc)
- 포트번호: 3000 (vite.config.ts)
- 10px = 1rem (reset.css)
